# Py-CY7C6521x
Python interfacing with GPIO of custom CY7C65215 board in macOS.  

https://www.cypress.com/file/129956/download


C-Python ffi from
https://github.com/tai/python-ucdev


### Build driver

```sh
brew install libusb
cd driver/library
sudo make
```
### Python dependencies  
```sh
pip3 install cffi
```
